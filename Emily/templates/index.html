COVID-19 Analysis Project Summary:

How has COVID-19 impacted the U.S.? Let's use data to find out. A Python script was written to analyze over 500,000 datapoints. This data includes the COVID-19 cases and deaths from January 21st to October 3rd, 2020. The data is reported daily by county by state.

Once the data was gathered, a spike map was created to show COVID-19 cases and deaths across the U.S. Also, a line graph was created for additional visualization. Click the Charts drop down for map selection.
